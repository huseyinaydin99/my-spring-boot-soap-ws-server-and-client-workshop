# my-spring-boot-soap-ws-server-and-client-workshop
Spring Boot ile SOAP Web Servisleri çalışmamdır. Bir SOAP Web Servisi sunucusu bir tanede SOAP istemcisi, Postman ve SOAPUI ile birlikte kullandığım projemdir. SOAP(Simple Object Access Protocol) ne kadar eskide olsa banka gibi yerlerde kullanılmakta ve her programcının eskiye yönelik desteği olması açısından bilmesi şarttır.
